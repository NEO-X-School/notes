# Menggunakan uv untuk Mengelola Environment dan Paket dari Python

Penulis: Dr. Bambang Purnomosidi D. P. (https://linktr.ee/bpdp)
PT Neo Akselerasi Indonesia
Last update: 6 Februari 2026

Peranti [uv](https://docs.astral.sh/uv/) merupakan peranti yang digunakan untuk mengelola instalasi Python, envs, serta paket. Dokumen ini menampilkan petunjuk praktis, bukan petunjuk lengkap. Petunjuk ini bisa digunakan secukupnya untuk membuat direktori (sering disebut dengan workspace) yang spesifik untuk suatu proyek tertentu dengan versi Python tertentu dan paket Python khusus untuk workspace tersebut.

## Instalasi

Petunjuk lengkap instalasi ada di https://docs.astral.sh/uv/getting-started/installation/. Sesuaikan dengan sistem operasi yang digunakan. Berikut adalah contoh di Linux:

```bash
$ curl -LsSf https://astral.sh/uv/install.sh | sh
downloading uv 0.9.30 x86_64-unknown-linux-gnu
no checksums to verify
installing to /home/bpdp/.local/share/../bin
  uv
  uvx
everything's installed!
WARN: The following commands are shadowed by other commands in your PATH: uv uvx
$
```

Setelah mengatur env variables untuk uv dan melakukan proses sourcing seperti petunjuk di atas (source $HOME …….. dst), uv  bisa dijalankan sebagai berikut:

```bash
$ uv
```

## Update

Update diperlukan jika menginginkan tetap menggunakan versi terbaru dari uv. Berikut adalah caranya:

```bash
$ uv self update
info: Checking for updates...
success: You're on the latest version of uv (v0.9.30)
$
```

## Membuat *Environment*

Berikut langkah untuk membuat environment:

```bash
$ uv venv
Using CPython 3.14.3+freethreaded
Creating virtual environment at: .venv
Activate with: source .venv/bin/activate
~/workspace-01> source .venv/bin/activate
(workspace-01) ~/workspace-01> 
$
```

**Perhatikan**: pada bagian depan terdapat `(workspace-01`. Tulisan tersebut menunjukkan workspace yang sedang aktif.

Jika ingin keluar dari env, gunakan perintah `deactivate`

```bash
(workspace-01) ~/workspace-01> python
Python 3.14.3 free-threading build (main, Feb  3 2026, 22:55:28) [Clang 21.1.4 ] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> 
(workspace-01) ~/workspace-01> deactivate 
~/workspace-01> python
 python: command not found
~/workspace-01> 
``` 

## Mengelola Paket

Untuk mengelola paket, gunakan `uv pip`. Perintah ini persis sama dengan `pip` di Python seperti biasanya tetapi dikhususkan untuk \textit{workspace} tersebut.

```bash
(workspace-01) ~/workspace-01> uv pip install pandas
Resolved 4 packages in 4.48s
Installed 4 packages in 3.43s
 + numpy==2.4.2
 + pandas==3.0.0
 + python-dateutil==2.9.0.post0
 + six==1.17.0
(workspace-01) ~/workspace-01>
```

Setelah selesai instalasi:

```bash
(workspace-01) ~/workspace-01> uv pip list
Package         Version
--------------- -----------
numpy           2.4.2
pandas          3.0.0
python-dateutil 2.9.0.post0
six             1.17.0
(workspace-01) ~/workspace-01> uv pip freeze > requirements.txt
(workspace-01) ~/workspace-01> cat requirements.txt 
numpy==2.4.2
pandas==3.0.0
python-dateutil==2.9.0.post0
six==1.17.0
(workspace-01) ~/workspace-01>
```
